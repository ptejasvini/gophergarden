## Problem Description
Given two variables, swap their values without using a temporary variable.

## Examples

**Example 1:**

Input: a = 5, b = 10
Output: a = 10, b = 5

**Example 2:**

Input: a = -3, b = 7
Output: a = 7, b = -3

**Example 3:**

Input: a = 0, b = 0
Output: a = 0, b = 0

## Constraints

- -10^9 <= a, b <= 10^9
- Variables can be of any numeric type
