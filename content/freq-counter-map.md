## Problem Description
Given a string, find the frequency of each character in the string.

## Examples

**Example 1:**

Input: str = "hello"
Output: {h: 1, e: 1, l: 2, o: 1}

**Example 2:**

Input: str = "aabbcc"
Output: {a: 2, b: 2, c: 2}

**Example 3:**

Input: str = ""
Output: {}

## Constraints

- String length: 0 <= n <= 10^5
- Characters can include any ASCII letters
