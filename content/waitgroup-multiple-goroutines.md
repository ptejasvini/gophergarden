## Problem Description
Write a Go program that launches 3 goroutines, each printing a message ("Goroutine 1", "Goroutine 2", "Goroutine 3"). Use sync.WaitGroup to ensure the main function waits until all goroutines finish before exiting.

## Examples

**Example 1:**

Input: NA
Output: 
Goroutine 3
Goroutine 1
Goroutine 2

## Constraints
- NA
